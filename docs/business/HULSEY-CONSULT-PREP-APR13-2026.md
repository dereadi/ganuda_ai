**Current claim:** Four-phase SRE protocol with architecturally distinct valence evaluation phase that operates on system actions before commit.

**Proposed broadening:** The valence evaluation phase reads not just system outputs but **inference-time weight-state deltas** as a real-time cognitive intensity signal. Specifically: the Frobenius norm of `dw` per chunk per layer (where `dw` is the per-chunk fast-weight update from a TTT-style mechanism) is a continuous, deterministic, near-zero-cost signal of "how much is the model updating itself right now," and the valence eval phase can use it as a structural correlate of decision intensity.

**Hulsey question:** Can Patent #3's claim language be broadened to cover "valence evaluation operating over inference-time weight-state changes" without losing the existing claim coverage? Is this a non-provisional broadening we should make in the conversion window (March 8 2027 deadline)?

**Why it's defensible:** The In-Place TTT paper publishes the mechanism without any audit/governance over the state deltas. Our claim covers the *governance* of those deltas, not the deltas themselves. Clean differentiation from prior art.

### Patent #4 (Graduated Autonomy Tiers for Multi-Timescale AI Systems) — broadening question

**Current claim:** Tiered autonomy framework (reflex, deliberation, council vote, longhouse ratification) for multi-agent AI systems.

**Proposed broadening:** Add a new top tier — **self-modification within bounded inference**. TTT-style mechanisms where the model rewrites its own weights within a single inference run constitute a structurally distinct kind of autonomy from tool use, multi-step reasoning, or external memory access. As more models adopt fast-weight mechanisms, the federation's tiering becomes the standard way to think about "when is it safe to let a model rewrite itself."

**Hulsey question:** Is "self-modification within bounded inference" a defensible top tier addition to the existing claim? How does it interact with the existing reflex/deliberation/council/longhouse tiering — does it sit above all of them, or only above reflex and deliberation?

**Why it matters commercially:** It positions the federation's graduated autonomy framework as the natural governance layer above any future TTT-style architecture. Patents that cover "the tier above where the industry is going" age well.

### Patent #2 (Sycophancy Detection in AI Agent Collectives) — research question (not a broadening yet)

**The new threat class:** TTT-induced sycophancy. If a model absorbs user framing into W_down via the per-chunk Hebbian update, repeated prompts in the same session may converge toward the user's worldview at the *weight* level. Patent #2's diversity check operates at the *output* level — it can detect that two specialist outputs are too similar, but it cannot detect that the model's W_down has been pulled toward the user's framing.

**Hulsey question (lighter weight):** Should Patent #2's claim language be broadened to cover weight-state similarity, or is this a separate patentable invention worth a new provisional? No urgency on this one — the threat class needs investigation first. **Flag it but don't lead with it.**
